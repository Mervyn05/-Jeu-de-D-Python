# 🎲 Jeu de Dé — Python

Jeu de dé en console avec deux modes : Solo (accumulation de points avec risque) et Duel (deux joueurs s'affrontent).

## 🎮 Modes de jeu

- **Solo** : Lance le dé et cumule les points. Si tu fais 1, tu perds tout !
- **Duel** : Deux joueurs lancent chacun N fois, le plus grand score gagne.

## ▶️ Lancer le projet

```bash
python jeu_de.py
```

## 💡 Concepts Python utilisés

| Concept | Utilisation |
|---|---|
| `random.randint()` | Lancer le dé |
| Fonctions | `lancer_de()`, `afficher_de()`, `mode_solo()`... |
| Boucles `while` / `for` | Manches et lancers |
| Dictionnaires | Stocker les scores |
| Gestion d'exceptions | Saisie invalide |
